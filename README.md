# Сайт строительной компании — инструкция по запуску

## Что внутри
- `index.html` — главная (лендинг с формой расчёта)
- `works.html` — наши работы / примеры домов
- `contacts.html` — контакты + форма
- `privacy.html` — политика конфиденциальности (шаблон)
- `oferta.html` — информация об условиях услуг (заготовка)
- `style.css`, `script.js` — стили и скрипты
- `assets/` — SVG-заглушки проектов (заменить на фото)

## Шаг 1. Замените заглушки (поиск по файлам)
| Что искать | На что заменить |
|---|---|
| `[НАЗВАНИЕ КОМПАНИИ]` и `[НАЗ<b>ВАНИЕ]` | ваше название (в лого вторая половина выделена цветом) |
| `+79000000000` и `+7 (900) 000-00-00` | ваш телефон |
| `[РЕГИОН]`, `[НАСЕЛЁННЫЙ ПУНКТ]`, `[ГОД]` | ваш регион, города, год начала работы |
| `[ФАМИЛИЯ ИМЯ ОТЧЕСТВО]`, `[ВАШ ИНН]` | реквизиты самозанятого (обязательно для Директа) |
| `info@вашдомен.ru`, `[АДРЕС САЙТА]`, `[ДАТА]` | почта, домен, дата публикации |
| `https://t.me/ВАШ_НИК` | ваш Telegram |
| `assets/project-*.svg` | реальные фото домов (webp/jpg, 200–400 КБ) |
| Цены `от 4 200 000 ₽` и т.д. | ваши реальные цены «от» |

## Шаг 2. Подключите приём заявок (Formspree, бесплатно)
1. Зарегистрируйтесь на formspree.io → New form → скопируйте ID вида `xabcdefg`.
2. Замените `ВАШ_ID` в `index.html` и `contacts.html` (атрибут `action` форм).
3. Заявки будут приходить на почту. Альтернатива — Telegram-бот, могу настроить.

## Шаг 3. Яндекс.Метрика
1. Создайте счётчик на metrika.yandex.ru, включите Вебвизор.
2. Вставьте код счётчика в `<head>` всех трёх основных страниц (место помечено комментарием).
3. В `script.js` замените `00000000` на номер счётчика.
4. В Метрике создайте цели с идентификаторами `form_submit` и `phone_click` (тип «JavaScript-событие») — они уже вшиты в код. Эти цели укажете в Директе для оптимизации.

## Шаг 4. Публикация на GitHub Pages
1. Создайте публичный репозиторий, например `site`.
2. Загрузите все файлы в корень репозитория (Add file → Upload files).
3. Settings → Pages → Source: Deploy from a branch → Branch: `main`, папка `/ (root)` → Save.
4. Через 1–2 минуты сайт откроется по адресу `https://ВАШ_ЛОГИН.github.io/site/`.

## Шаг 5. Свой домен (нужен для Директа)
1. Купите домен (reg.ru, nic.ru и т.п.).
2. В настройках DNS домена создайте A-записи на IP GitHub Pages:
   `185.199.108.153`, `185.199.109.153`, `185.199.110.153`, `185.199.111.153`
   и CNAME `www` → `ВАШ_ЛОГИН.github.io`.
3. В репозитории: Settings → Pages → Custom domain → введите домен → дождитесь проверки → включите Enforce HTTPS.

## Перед запуском Директа — чек-лист
- [ ] ФИО и ИНН самозанятого в подвале (уже заложено — просто заполнить)
- [ ] Политика конфиденциальности заполнена, чекбокс согласия в формах (есть)
- [ ] Цены на сайте совпадают с ценами в объявлениях
- [ ] Метрика стоит, цели созданы
- [ ] Телефон кликабелен на мобильном (есть, плюс плавающая кнопка звонка)
- [ ] Сайт открывается по своему домену с HTTPS

## Юридическая пометка
Тексты политики и оферты — шаблоны-заготовки. Перед публикацией сверьте политику с вашим уведомлением в РКН; при сомнениях покажите юристу.
